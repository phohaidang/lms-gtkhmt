Hoạt động 2.2 (30 phút) — Hàm đệ quy đầu tiên: giai thừa

  • Định nghĩa toán học: n! = n * (n-1)!, với 0! = 1.
  • Code: python def giai_thua(n): if n == 0: return 1 # base case (điều kiện dừng) return n * giai_thua(n-1) # recursive case
  • Quy tắc 2 phần của đệ quy:
  • Base case (điều kiện dừng) — nếu thiếu → đệ quy vô hạn → tràn stack.

  • Recursive case — phải tiến gần đến base case (n giảm về 0).

  • Demo "trace": Vẽ trên bảng quá trình giai_thua(4) → 4 * giai_thua(3) → 4 * 3 * giai_thua(2) → ... → kết quả 24.

  • Bài tập: Viết hàm đệ quy tính tong(n) = 1 + 2 + ... + n.

Hoạt động 2.3 (20 phút) — Fibonacci & cảnh báo hiệu năng

  • Code: python def fib(n): if n <= 1: return n return fib(n-1) + fib(n-2)
  • Demo cảnh báo: Chạy fib(30) — vài giây. Chạy fib(40) — chờ rất lâu. Vì sao? (Buổi 6 sẽ phân tích — cài cắm cho buổi sau.)
